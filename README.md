# movie-recomendation-system-using-cosinesimilarity

🎬 Movie Recommendation SystemA Machine Learning-based recommendation engine that suggests movies to users based on content similarity. This project utilizes Natural Language Processing (NLP) and Cosine Similarity to find the closest matches to a user's favorite film.📌 Table of ContentsOverviewRecommendation TypesDatasetWorkflowDependenciesTechnical Explanation📖 OverviewThe goal of this project is to build a Content-Based Recommendation System. By analyzing metadata such as movie titles, cast, and crew, the system calculates the mathematical similarity between films and provides a list of top-tier recommendations.🤖 Recommendation TypesThere are three primary ways to build a recommendation engine:Content-Based Filtering: Suggests items similar to what a user has liked before (based on attributes like genre, director, or description).Popularity-Based Filtering: Suggests items that are currently trending or have the highest average ratings across the entire user base.Collaborative Filtering: Suggests items based on the behavior of similar users ("Users who liked X also liked Y").📊 DatasetThe system uses a dataset containing over 4,800 movies. The key features used for building the recommendation engine are:title: The name of the movie.cast: The lead actors in the film.crew: The director and production members.🛠 WorkflowThe project follows these logical steps:Data Collection: Loading the movies.csv file into a Pandas DataFrame.Feature Selection: Identifying the most relevant columns (title, cast, crew) to define a movie's "personality."Data Preprocessing: Combining the selected text features into a single string for each movie and handling null values.Vectorization: Converting text data into numerical values using TfidfVectorizer.Similarity Computation: Calculating the Cosine Similarity score between all movie vectors.User Interaction: Taking a movie name from the user and using difflib to handle typos before suggesting matches.📦 DependenciesTo run this project, you need the following Python libraries:numpy: For numerical operations.pandas: For data manipulation and analysis.difflib: To find the closest match for user input.scikit-learn: For the TfidfVectorizer and cosine_similarity algorithms.🔬 Technical Explanation1. TF-IDF VectorizerSince machines cannot understand raw text, we use Term Frequency-Inverse Document Frequency (TF-IDF). It converts text into a matrix of numbers where higher weights are given to words that are unique and descriptive.2. Cosine SimilarityTo determine how "close" two movies are, we calculate the cosine of the angle between their two vectors.A score of 1.0 means the movies are identical.A score of 0.0 means they have no similarity.$$\text{similarity} = \cos(\theta) = \frac{\mathbf{A} \cdot \mathbf{B}}{\|\mathbf{A}\| \|\mathbf{B}\|}$$
